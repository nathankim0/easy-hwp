# 한글 파일 분석

## 입력
$ARGUMENTS

## 해야 할 일

1. **HWPX 파일 열기**: ZIP 형식이므로 압축 해제하여 `Contents/section*.xml` 확인
2. **표 구조 파악**: XML에서 표(`tbl`), 행(`tr`), 셀(`tc`) 찾아서 내용 추출
3. **필드 식별**: 2열 표에서 첫 번째 열은 필드명, 두 번째 열은 값으로 인식
4. **빈 필드 구분**: 값이 비어있는 필드 = 채워야 할 항목

## 사용자에게 알려줄 것

- 발견된 표 개수
- 채워야 할 필드 목록 (빈 칸)
- 이미 채워진 필드 (있다면)
- 다음 단계 안내 (`/hwp-fill`로 채우기)

## 주의

- `.hwp`는 지원 안 됨 → 한글에서 `.hwpx`로 다시 저장하라고 안내
- 복잡한 표(병합 셀 등)는 구조가 다르게 보일 수 있음
- 기술적 에러는 사용자가 이해할 수 있게 설명
